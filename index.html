CI/CD Pipeline Demo

Created by Adithya

This project demonstrates how a simple static website can be automatically validated, checked, built, and deployed using a modern DevOps CI/CD pipeline powered by GitHub Actions.

✔ Git Workflow Used
  • Create Repository
  • Create develop branch
  • Add Files & Make Code Changes
  • Commit & Push
  • Create Pull Request (develop → main)
  • Merge PR
✔ CI/CD Pipeline Stages
  • Stage 1 — Code Checkout
  • Stage 2 — Syntax Validation
  • Stage 3 — Link Checking
  • Stage 4 — Build Preview
  • Stage 5 — Deployment to GitHub Pages

After these steps are completed, GitHub Pages hosts this website live using automated workflows.
